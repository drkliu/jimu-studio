- Entity create is explicit, version-zero, duplicate-safe, bounded to 200 dynamic fields, keyboard/label checked, and provider-authoritative.

- A non-deletable dependency plan renders its dependencies without retaining an applicable draft.
- Conflict and replay invalidate the draft and never silently retry deletion.
- The native local Provider runs on loopback only, exercises every role mutation without Docker, and appends a bounded audit record for every mutation.
- Audit, usage, and discrepancies remain intentionally read-only because the contract defines no mutations.
