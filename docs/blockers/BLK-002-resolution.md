# BLK-002 resolution

Resolved by the requirement owner on 2026-07-16: use option 2.

Studio release 1.0.0 will provide entity/field visualization and guarded entity editing. Relation visualization is removed from S2 acceptance and deferred until a future provider-owned, versioned contract defines relation identity, source, target, cardinality, pagination, mutation concurrency, and migration behavior. No relation semantics will be inferred from field names or data types.
