# Jimu Studio contract v1

This directory is generated by `jimuctl studio export`. Do not edit generated files.

- Providers bind the OpenAPI operations to the corresponding Jimu APIs and retain deny-by-default runtime authorization.
- Clients use `client.ts` with an opaque OIDC bearer token. Never send trusted tenant or user headers.
- Tenant switching requires a fresh tenant-scoped token and a fresh client instance.
- CI must run `jimuctl studio verify --dir <this-directory>` against the pinned Jimu release before typecheck/browser tests.
